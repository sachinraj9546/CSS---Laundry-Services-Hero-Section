# Laundry Services Landing Page

A responsive landing page for a laundry service built using HTML and CSS. This project was created to practice responsive web design, Flexbox, media queries, and building a mobile navigation menu without JavaScript.

## Features

- Responsive navigation bar
- Mobile hamburger menu
- CSS-only menu toggle
- Hero section with call-to-action button
- Responsive image section
- Button hover animations
- Mobile, tablet, and desktop support

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Roboto, Lato)

## Project Overview

This project consists of a modern landing page for a laundry service. The navigation bar automatically adapts to different screen sizes. On larger screens, users can access navigation links directly, while on smaller screens a hamburger menu is displayed.

The layout is built using Flexbox and media queries to ensure a smooth user experience across devices.

## What I Learned

- Responsive Web Design
- CSS Flexbox
- Media Queries
- CSS Positioning
- CSS Transitions and Animations
- Building a Hamburger Menu using CSS

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder.
3. Open `index.html` in your browser.
